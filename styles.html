                Multiple Background Images

                RGBA = Red Green Blue Alpha
                color: rgb(6, 68, 0, 0.354); 
                Sets the transparency of colors
                0 alpha is invisible
                1 alpha is fully visible
            
                Opacity
                opacity: 0.25;
                Sets the transparency of elements
                0 is invisible
                1 is fully visible

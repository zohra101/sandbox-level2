Image Map



Image maps allow you to create clickable hotspots on images and use them as hyperlinks, by selecting an area of the image and mapping its coordinates, and then adding the href tag to the destination. I thought these fell out of disfavor for accessbility reasons, because it's hard for a user to identify that part of an image is a hotspot.


  1. Use map and area to define an image map
    • map needs the name attribute
    • img needs the usemap attribute
    • usemap should start with # and match name
  2. area needs the shape attribute
    • shape can be rect, circle, or poly
    • area needs the coords attribute
    • coords can be obtained with Luna Paint or Pix Spy
  3. add href to area to make it clickable
  4. add title to area for tooltip popups
  5. add target="_blank" to area so links open in a new tab
  6. add link to image resizer JSDelivr CDN
    • Call imageMapResize() to automatically scale image maps

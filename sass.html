SASS

Syntactically Aswesome Style Sheet

                SASS nesting syntax
                pre {
                    code {
                        color:red;
                    }   
                }
                A style can be nested in another style to select a nested element.
                The code tag in the pre tag is styled.
            
                $myColor: purple;
                pre{ 
                     i {
                    color:$myColor;
                    }
                }

                SASS variables start with $.
                Use colon to store a value.
                Use the variable name to access the value.
            

                Use SASS to remember repeats

                Background image variable
                -------------------------
                $thumbsUp:url("thumbs-up.jpg")
                .bg-image {
                    background-image: $thumbsUp;
                }
                

                Horizontal repeat variable
                --------------------------
                $horizontalRepeat: space no-repeat;
                .horizontalRepeat {
                     background-image: $horizontalRepeat;
                 }
                

                Veritcal repeat variable
                ------------------------
                $verticalRepeat: no-repeat repeat;
                .verticalRepeat {
                background-repeat: $verticalRepeat;
                }
                

            
                Use SASS to remember sizes

                Background size variable
                ------------------------
                background-size: 25% 15%;
                $stretch: 25% 100%
                .stretch {
                background-size: $stretch;
                }
            
                Use SASS to remember Color Stop Gradients

                Gradient variable
                ------------------------
                $darkGradient: linear-gradient(90deg, gray, black 10% 90%, gray);
                $lightGradient: linear-gradient(90deg, gray, white 10% 90%, gray);
                .gradient {
                background-image: $darkGradient;
                }
            
                RGBA Color variables
                --------------------
                $darkRed: rgba(120, 0, 0, 1);
                $darkGreen: rgba(2, 53, 2, 1);
                $lightBlue: rgb(0, 0, 255, 1);
                1. Declare the color variables all in one area. 
                2. Use them throughout the SASS file.
                3. These variables are the single source of truth, so changing the value of the variable affects the whole file.
            
                Single line comments
                ---------------------------
                //Custom colors
                $darkRed: rgba(120, 0, 0, 1);
                $darkGreen: rgba(2, 53, 2, 1);
                $lightBlue: rgb(0, 0, 255, 1);
                Like JavaScript, SASS supports single line comments (unlike CSS).
            
                Adjust width to automatically fit content
                -----------------------------------------
                width: fit-content
                Useful on block elements.
                Block elements will still use a whole line, but the hit box of the block element is reduced.
                Compare it with display: inline, which lines up the elements. 
            
                Parent syntax
                ------------------
                &;hover
                Unique to SASS, groups styles together as children of other styles. 
                h1 {
                    color: green;
                    width: fit-content;
                
                    &:hover {
                        color: $darkRed
                    }
                }

                Mixins are like variables
                A single-line mixin stores a property and value.
                Type @mixin to declare/define a mixin variable.
                Type @include to use the mixin.
                Mixins can substitute for utility classes so elements can be described by their name.
